# DERECHO DE EXPORTACION REDUCCION DE LA ALICUOTA PARA DETERMINADAS MERCADERIAS DE CONSUMO

<a id="1"></a>
ARTICULO  1.- La alícuota del derecho de exportación adicional que grava a las  exportaciones  para  consumo  de  las  mercaderías comprendidas  en  las  posiciones  arancelarias  de la NOMENCLATURA ARANCELARIA Y DERECHOS DE EXPORTACION (N.A.D.E.) detalladas  en  el Anexo  al  presente  decreto,  el  que  forma  parte integrante del mismo,  se  reduce a los niveles y a partir de las  fechas  que  en cada caso se indica.

<a id="2"></a>
ARTICULO  2.-  Lo  dispuesto  en  este  decreto  resultará  de aplicación:

a)  Para  productos comprendidos en la Ley 21.453 desde la fecha de su vigencia  y  para  embarques  que  se  realicen  a partir de las fechas que para cada caso se indican.

b)  Para  los  productos  no  comprendidos  en la Ley 21.453,  para embarques  que  se realicen a partir de las fechas  que  para  cada caso se indican.

<a id="3"></a>
ARTICULO 3.- El presente decreto entrará en vigencia el día de su publicación en el Boletín Oficial.

<a id="4"></a>
ARTICULO  4.-  Comuníquese,  publíquese,  dése  a la Dirección Nacional del Registro Oficial y archívese.
