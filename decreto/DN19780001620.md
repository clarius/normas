# DECRETO SECRETO SOBRE DENEGACION PARA SALIR DEL PAIS

<a id="1"></a>
Artículo 1º - Deniégase la solicitud para que se le permita salir del país a la detenida a disposición del Poder Ejecutivo Nacional Mabel Adriana GRINBERG (L.C. 5.806.264).

<a id="2"></a>
Art. 2º - Comuníquese, cúmplase y ARCHIVESE.
