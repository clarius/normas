# DESIGNACION CON EL NOMBRE "DOCTOR MARIANO MORENO" A LA BIBLIOTECA NACIONAL.

<a id="1"></a>
ARTICULO 1º - Desígnase con el nombre "Doctor Mariano Moreno" a la Biblioteca Nacional.

<a id="2"></a>
ARTICULO 2º - Comuníquese al Poder Ejecutivo nacional.
