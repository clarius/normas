# FACULTADES DEL PODER EJECUTIVO PARA EL ORDENAMIENTO LEGISLATIVO.

<a id="1"></a>
ARTICULO  1.-  Facúltase  al Poder Ejecutivo para ordenar las leyes, sin introducir  en  su  texto    ninguna  modificación,  salvo  las gramaticales indispensables para la nueva ordenación.

<a id="2"></a>
ARTICULO  2.-  Comuníquese, publíquese, dése a la Dirección Nacional del Registro Oficial y archívese.
