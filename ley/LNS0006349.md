# Deja sin efecto el requisito de edad para el otorgamiento de las becas de investigación del Conicet, para las becas postdoctorales y para el ingreso a la carrera de investigador

<a id="1"></a>
ARTÍCULO 1°.- Déjase sin efecto el requisito de edad para el otorgamiento de las becas de investigación del CONICET, como así también para las becas postdoctorales y el ingreso a la carrera de investigador.

<a id="2"></a>
ARTÍCULO 2°.- Comuníquese al Poder Ejecutivo nacional.
