# APROBACION DE UN ACUERDO PARA LA EJECUCION DE EXHORTOS CON PARAGUAY.

<a id="1"></a>
ARTICULO 1.- Apruébase el acuerdo de 31 de Agosto del corriente año, celebrado entre  los  Plenipotenciarios  de la República Argentina y el de Paraguay para la ejecución de exhortos.

<a id="2"></a>
ARTICULO 2. Comuníquese al Poder Ejecutivo.
