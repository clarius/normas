# Declaración del Día de las Futbolistas Argentinas el 21 de agosto de cada año

<a id="1"></a>
Artículo 1°- Declárese Día de las Futbolistas Argentinas el 21 de agosto de cada año.

<a id="2"></a>
Art. 2°- Comuníquese al Poder Ejecutivo nacional.
